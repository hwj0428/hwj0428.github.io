Kali 虚拟机安装及网络配置详细指南

一、准备工作

  1. 下载 Kali Linux 镜像：前往 Kali Linux 官方网站（https://www.kali.org/downloads/ ），根据你的需求选择合适的 Kali Linux 镜像版本，推荐下载最新的稳定版。

  2. 安装虚拟机软件：本文以 VMware Workstation 为例，你可以从 VMware 官方网站下载并安装该软件。安装过程中按照默认设置即可，若有特殊需求可根据提示进行调整。

二、创建 Kali 虚拟机

  1. 打开 VMware Workstation，点击 “创建新的虚拟机”。

  2. 在 “新建虚拟机向导” 中，选择 “典型（推荐）”，然后点击 “下一步”。

  3. 选择 “安装程序光盘映像文件（iso）”，浏览并选择之前下载好的 Kali Linux 镜像文件，点击 “下一步”。

  4. 在 “虚拟机名称” 处输入你想要的名称，如 “Kali VM”，并选择虚拟机的存储位置，点击 “下一步”。

  5. 根据你的计算机硬件配置，选择合适的磁盘大小。建议至少分配 20GB 的磁盘空间，若有更多需求可适当增加。选择 “将虚拟磁盘存储为单个文件”，然后点击 “下一步”。

  6. 点击 “完成”，虚拟机创建完成。

三、安装 Kali Linux 系统

  1. 在 VMware Workstation 中，选中刚创建的 Kali 虚拟机，点击 “开启此虚拟机”。

  2. 进入 Kali Linux 安装界面，选择安装语言，如 “English”，然后点击 “继续”。

  3. 选择区域设置，如 “United States”，点击 “继续”。

  4. 选择键盘布局，如 “English (US)”，点击 “继续”。

  5. 在 “配置网络” 步骤，确保网络连接正常，若未自动获取到 IP 地址等信息，可手动进行配置。一般情况下，保持默认设置让其自动获取即可。点击 “继续”。

  6. 设置主机名，如 “kali - vm”，点击 “继续”。

  7. 设置域名，若没有特殊需求可留空，点击 “继续”。

  8. 设置 root 用户密码，输入两次相同的密码，点击 “继续”。

  9. 磁盘分区选择 “自动分区”，然后点击 “继续”。系统会提示分区更改，确认无误后点击 “继续”。

  10. 等待安装过程完成，安装完成后点击 “立即重启”。

四、配置 Kali 虚拟机网络

  1. NAT 模式（推荐）：

  • 打开 VMware Workstation，点击 “编辑” -> “虚拟网络编辑器”。

  • 在虚拟网络编辑器中，选择 “NAT 模式”，确保 “使用本地 DHCP 服务将 IP 地址分配给虚拟机” 选项被勾选。点击 “确定”。

  • 进入 Kali 虚拟机系统，打开终端，输入命令 “ifconfig”，查看是否获取到了有效的 IP 地址。若未获取到，可尝试重启网络服务，命令为 “service networking restart”。

  • 测试网络连接，在终端输入 “ping www.baidu.com”，若能正常 ping 通，说明网络配置成功。

  1. 桥接模式：

  • 同样在 VMware Workstation 的 “虚拟网络编辑器” 中，选择 “桥接模式”，并在 “桥接到” 下拉菜单中选择你物理机所使用的网络适配器，点击 “确定”。

  • 进入 Kali 虚拟机系统，编辑网络配置文件。在终端输入 “nano /etc/network/interfaces”，打开文件后，内容可修改为：
    auto eth0iface eth0 inet dhcp

  • 保存并退出文件（按 “Ctrl + X”，然后按 “Y”，最后按 “Enter”）。

  • 重启网络服务，命令为 “service networking restart”。

  • 再次使用 “ifconfig” 命令查看 IP 地址，并用 “ping” 命令测试网络连接。
